# Healthcare policy pack (template)

`policy-pack.json` in this folder is a **starter** vertical pack: it lists compliance rule **keys** and advisory defaults you can promote into a versioned policy pack through the operator UI or API.

- Do not treat this file as a legal HIPAA attestation — it is an architecture-governance starting point.
- Pair with [`docs/go-to-market/HEALTHCARE_VERTICAL_BRIEF.md`](../../go-to-market/HEALTHCARE_VERTICAL_BRIEF.md) and [`docs/trust-center.md`](../../trust-center.md) for buyer-facing boundaries (no PHI in briefs).
